> round함수의 사사오입(Round half to Even)
>> * round(2.5)는 2입니다. (가장 낮은 자리 숫자가 5이므로 짝수로 반올림)
>> * round(3.5)는 4입니다. (가장 낮은 자리 숫자가 5이므로 짝수로 반올림)
>> * round(1.5)는 2입니다. (가장 낮은 자리 숫자가 5이므로 짝수로 반올림)
>> * round(2.75, 1)은 2.8입니다. (소수점 한 자리까지 반올림)
